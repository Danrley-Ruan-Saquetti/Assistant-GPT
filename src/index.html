# About App

   The GPT Assistant is an app focused on dialogue, used to answer the most varied questions, whether they are about problem issues, solving exercises or any other type of help, both personal and professional.

# Instructions

   To use it, it is necessary to enter the settings and inform the API Key, available on the official OpenAI platform itself, in addition to other settings, such as the maximum number of characters and the creativity rate of responses, which varies between 0 (zero) and 2 (two).
